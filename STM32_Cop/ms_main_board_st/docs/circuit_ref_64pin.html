메가소닉 메인보드 회로 참조 (64pin 재편성)

대상: STM32G474RETx LQFP64 기반 메가소닉 메인보드
기준: 기능 우선 + 아트웍 편의(물리적 핀 군집화)
레거시 80핀 문서: docs/circuit_ref_80pin.md

1. 문서 목적

이 문서는 기존 80핀 배치에서 64핀 패키지로 전환할 때 필요한 회로/핀맵 재편성 기준을 정리한다.
핵심 목표는 다음 2가지다.

  • 기능 유지: HRTIM, ADC, RS-485, LCD/LED 래치, 알람/릴레이, 외부입력 기능을 동일하게 유지
  • 아트웍 최적화: 신호 성격별로 핀을 가능한 한 같은 포트/인접 군으로 배치해 배선 단순화

2. 64pin 전환 핵심 요약

2.1 변경 핵심 (필수 재배치)

기능 80pin 64pin 변경 이유
LCD 래치 클럭 PD8 PC14 LQFP64에서 PD8 미사용, GPIOC 상단 핀으로 재배치
LED 래치 클럭 PD9 PC15 LQFP64에서 PD9 미사용, GPIOC 상단 핀으로 재배치
SENSOR 입력 PD0 PD2 외부입력의 독립 라인 확보 및 PC13 여유 핀 회수
LOW_ALARM PC15 PA10 BOOT0 핀 인접 충돌 회피 및 알람 출력 안정화
BUZZER PB5 PC5 디스플레이/외부입력 라우팅 단순화를 위한 재배치

2.2 여유 핀 변화

  • 64pin 재배치(클럭/알람 포함) 후 spare 재조정
  • BUZZER, SENSOR 재배치 반영 후 최종 여유 6개 유지

최종 spare:

  • PA5
  • PB2
  • PB5
  • PB8
  • PB9
  • PC13

3. 핀 재편성 원칙

3.1 기능 우선 원칙

  • 고속/타이밍 민감 신호는 유지
  • HRTIM PWM: PA8/PA9 유지
  • ADC 4채널: PA0/PA1/PA6/PA7 유지
  • RS-485: PA2/PA3 + PB0/PB1 유지

3.2 아트웍 편의 원칙

  • 디스플레이 버스 군집화:
  • PC14/PC15 + PB10~PB15로 통일
  • 외부 입력 군집화:
  • PC10/PC11/PC12 + PD2(SENSOR)
  • 릴레이 군집화:
  • LC 릴레이는 PC6~PC9 유지
  • 알람 릴레이는 PA10/PA11/PA12/PB3/PB4 유지

4.8.1 공진 튜닝 인덕턴스 조합표 (고정 + 릴레이 4개)

자동 공진 탐색에서 저인덕턴스(2MHz 근처) 분해능을 높이기 위해,
2진 가중(2/4/8/16uH) 대신 고정 인덕턴스 + 릴레이 조합 구조를 사용한다.

  • 고정 인덕턴스:
  • L_BASE = 3.3uH || 3.9uH = 1.79uH (근사)
  • 릴레이 분기 인덕턴스:
  • bit0 (LC_RELAY1, PC6): L1 = 1.5uH || 2.2uH = 0.89uH
  • bit1 (LC_RELAY2, PC7): L2 = 2.7uH
  • bit2 (LC_RELAY3, PC8): L3 = 4.7uH
  • bit3 (LC_RELAY4, PC9): L4 = 6.8uH

총 인덕턴스:

  • L_total = L_BASE + (bit0?L1:0) + (bit1?L2:0) + (bit2?L3:0) + (bit3?L4:0)
  • 범위: 1.79uH ~ 16.88uH (16조합)

주요 타깃 근사:

  • 1.76uH 목표: 0000 -> 1.79uH
  • 2.64uH 목표: 0001 -> 2.68uH
  • 16.9uH 목표: 1111 -> 16.88uH

4.8.2 C값별 공진 주파수 LUT (16조합 x 3종 C)

공진 주파수는 아래 식으로 계산한다.

  • f_res = 1 / (2pisqrt(L_total*C))
  • L_total: 4.8.1 조합표 값 (uH)
  • C: 트랜스듀서 정전용량 (1.5nF / 2.4nF / 3.6nF)

0001 2.68 2510.2 1984.5 1620.3
0010 4.49 1939.3 1533.2 1251.8
0011 5.38 1771.7 1400.6 1143.6

참고:

  • 펌웨어 기본 채널 운용 범위(500~2000kHz) 기준에서, C=1.5nF는 0010~1111 조합이 유효 대역에 들어온다.
  • C=2.4nF, 3.6nF는 0001~1111 조합 대부분이 유효 대역에 들어오며, 0000은 2MHz 초과 가능성이 높다.

5. 물리적 군집 관점 라우팅 가이드 (64pin)

아래 군집은 배선 난이도/길이/크로스토크를 줄이기 위한 권장 배치다.

5.1 디스플레이 군집 (하측/우하측 권장)

  • PC14, PC15, PB10~PB15
  • 20핀 커넥터를 MCU 인접에 배치하고, DATA+CLK를 한 면에서 바로 인출

5.2 외부입력 군집 (하측/좌하측 권장)

  • PA15, PC10~PC13
  • PA15, PC10~PC12, PD2
  • 광커플러(TLP281-4 + TLP181) 입력단을 같은 영역으로 묶어 GND 리턴 경로 단순화

5.3 전력/고속 군집

  • HRTIM: PA8/PA9는 UCC21520까지 최단거리
  • ADC: PA0/PA1/PA6/PA7은 아날로그 섬으로 분리
  • RS-485: PA2/PA3 + PB0/PB1는 트랜시버 근접 배치
