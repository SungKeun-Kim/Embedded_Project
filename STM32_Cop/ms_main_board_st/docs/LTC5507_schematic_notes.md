# LTC5507 (PZT FWD/REV 검출) 스키매틱 검토 메모

Analog Devices [LTC5507](https://www.analog.com/en/products/ltc5507.html) 데이터시트(5507f.pdf) 기준.

## 1. 데이터시트 대조 (PCAP, RFIN, SHDN)

| 항목 | 데이터시트 요지 | 권장 연결 |
|------|-----------------|-----------|
| RFIN (6) | RF 입력 | 커플러 측 **AC 커플링 캐패시터**(기존 C16/C17 등) 뒤 RFIN만 연결 |
| PCAP (5) | 피크 검출용 **외부 캐패시터** 단자 | **PCAP–VCC 사이**에 C_peak (예: 3.3nF, 주파수에 따라 조정). **RFIN과 동일 노드로 묶지 않음** |
| ~SHDN (1) | Low=셧다운, High=동작 | **VCC 또는 논리 High**로 고정. 부동 금지 |
| VCC (4) | 2.7~6V | 보드의 +5V 등 허용 범위 내 전원 |

## 2. 커플러 탭 → RFIN 입력 레벨 (검증 절차)

- 데이터시트: RFIN 허용 입력 전력 대역(대략 **약 -34 dBm ~ +14 dBm**, 조건은 주파수·온도별 표 참조).
- 도면상으로는 커플링·종단 네트워크만으로는 dBm을 확정할 수 없음.
- **권장**: 최대 출력 조건에서 스펙트럼 분석기 또는 스펙트럼 모니터로 FWD/REV 탭 레벨 측정 → LTC5507 표의 **최대 입력** 이하인지 확인. 필요 시 **고정 감쇠 패드** 추가.

공식(50Ω 시스템, RMS 전압 \(V\)로부터):

\[
P_{\mathrm{dBm}} = 10 \log_{10}\left(\frac{V^2 / 50}{1\,\mathrm{mW}}\right)
\]

## 3. KiCad 반영 (KI_Mega_PWBD)

- PCAP을 FWD/REV RF 버스에서 분리하고 **C18/C19 (3.3nF)** 로 PCAP–VCC 연결.
- **~SHDN**을 각 IC **VCC 노드**로 연결.
- U9 RFIN: **FWD** 측 101.6mm 그리드로 **419.1–381** 구간 배선 보강.
- U10 RFIN: **REV** 측 **454.66** 수직–**458.47** RFIN 배선 보강.
- VCC는 **+5V** 파워 심볼과 동일 네트로 연결 (LTC5507 VCC 범위 내).

KiCad에서 **ERC** 실행 후 미연결 핀이 있으면 동일 블록 내 GND/+5V를 맞춰 주면 됩니다.
