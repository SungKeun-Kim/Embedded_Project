# STM32G474RETx LQFP64 Bring-up 스모크 테스트

> 목적: 조립 직후 보드의 치명적 결함을 빠르게 배제하고, 펌웨어 디버깅 가능한 상태로 올리는 절차
> 기준: include/config.h, src/gpio_init.c, docs/cubemx_64pin_pin_table.md

---

## 0. 준비

- 장비:
  - 가변 전원공급기(전류 제한 가능)
  - DMM(멀티미터)
  - ST-Link
  - USB-UART 또는 RS-485 어댑터
  - 오실로스코프(권장)
- 펌웨어:
  - Debug 빌드 사용
  - 초기 출력은 LOW 유지 (안전 상태)

---

## 1. 전원 인가 전 점검 (무전원)

- [ ] 3.3V-GND 저항이 비정상 단락이 아닌지 확인
- [ ] 5V-GND 저항이 비정상 단락이 아닌지 확인
- [ ] HRTIM 출력(PA8/PA9)과 파워 스테이지 사이 쇼트가 없는지 확인
- [ ] RS-485 A/B, GND 배선 극성 확인
- [ ] 릴레이 코일 전원 라인 쇼트 확인

판정 기준:

- 즉시 0옴 수준 쇼트가 있으면 전원 인가 금지

---

## 2. 1차 전원 인가 (전류 제한)

권장 순서:

1. 전원공급기 전류 제한을 낮게 설정
2. 입력 전압을 목표 전압까지 천천히 상승
3. 과전류 트립 여부 확인

체크:

- [ ] 트립 없이 기동
- [ ] 3.3V 레일 정상
- [ ] 5V 레일 정상
- [ ] MCU 발열 이상 없음

판정 기준:

- 과전류 트립, 비정상 발열 발생 시 즉시 전원 차단 후 하드웨어 재점검

---

## 3. SWD 연결 및 기본 펌웨어 다운로드

```bash
cmake -B build -G "Ninja" -DCMAKE_BUILD_TYPE=Debug
cmake --build build
cmake --build build --target flash
```

체크:

- [ ] ST-Link가 타깃을 인식
- [ ] 다운로드 성공
- [ ] 리셋 후 재접속 가능

---

## 4. 기본 GPIO 스모크 테스트

### 4.1 버튼 입력 (PC0~PC4, Active Low)

- [ ] 무입력 상태에서 HIGH
- [ ] 버튼 누름 시 LOW
- [ ] 채널 오동작(다른 버튼 동시 반응) 없음

### 4.2 외부 입력 (PA15, PC10~PC12, PD2)

- [ ] 풀업 상태에서 기본 HIGH
- [ ] 강제 접지 시 LOW 전환
- [ ] SENSOR_INPUT이 PD2로 반영됨

### 4.3 릴레이/부저 출력

- [ ] RELAY: PA10/PA11/PA12/PB3/PB4 매핑 확인
- [ ] LC_RELAY: PC6~PC9 매핑 확인
- [ ] BUZZER: PC5 매핑 확인
- [ ] 부팅 직후 모두 LOW(안전 상태)

---

## 5. 통신 스모크 테스트

### 5.1 디버그 UART (USART1, PB6/PB7)

- [ ] 115200bps 수신 확인
- [ ] 부팅 로그/헬스 로그 확인

### 5.2 RS-485 (USART2, PA2/PA3, PB0/PB1)

- [ ] idle 상태에서 DE/RE 제어 정상
- [ ] 송신 시 DE 활성, 송신 완료 후 수신 모드 복귀
- [ ] 기본 Modbus 프레임 송수신 확인

---

## 6. HRTIM 출력 확인 (PA8/PA9)

사전 조건:

- 파워 스테이지 분리 또는 안전 부하 상태

체크:

- [ ] PA8/PA9 파형 출력 확인
- [ ] 상보 파형 관계 정상
- [ ] 주파수 변경 시 파형 추종
- [ ] 비상 정지 조건에서 출력 차단

주의:

- 공진/고전력 시험은 스모크 테스트 범위를 벗어나므로 별도 절차로 수행
